地图API接口文档说明
版本：3.0.0.0 作者:王德海
构造函数 配置方法 参数说明 返回值 说明
CqrmMap(divId,mapKey) 地图容器ID None 构造地图SDK初始类
SDK初始类 Map() 主要初始化地图和地图基本操作 Map类对象 实例化的地图类
Icon() 地图小图标类 Icon类对象 实例化的任意图标类
CarIcon() 车辆图标类 CarIcon类对象 实例化的单个车辆图标类
Line() 地图线路类 Line类对象 实例化的地图线路类
Fence() 地图多边形类 Fence类对象 实例化的地图多边形类
Circle() 地图圆形类 Circle类对象 实例化的地图圆形类
Rectangle() 地图多边形类 Rectangle类对象 实例化的地图多边形类
Map类 getCenter() None "lat,lng" 获取地图中心坐标
getBounds() None "lat,lng;lat,lng" 获取地图范围内的东北点和西南点
setCenter(lat,lng) lat纬度、lng经度 None 设置地图中心坐标
getZoom() None number 获取地图当前缩放级别
setZoom(number) int None 设置地图为指定缩放级别
setMaxZoom(number) int None 设置地图最大缩放级别
setMinZoom(number) int None 设置地图最小缩放级别
resize() None None 使地图适应容器变化
getPosition(lat,lng,backFun) 纬度、经度、回调函数 回调函数,内容为地点描述信息 获取指定点的地理位置信息
searchPosition(address,backFun) 地理位置描述,回调函数 回调函数,内容为"lat,lng" 通过地理位置获取第一个符合的点坐标
boxZoomInOrZoomOut(string) 'in'放大，'out'缩小 None 拉框放大，拉框缩小
getUserLocation(backFun) 回调函数 backFun(String) 获取用户地理位置经纬度
addEventListener(backFun) 回调函数 backFun(eventName,eventObj) 获取地图监听事件(onzoomend/ondragend/onresize)
pointToPixel(lat,lng) 经纬度 像素点(x,y) 经纬度转像素点
pixelToPoint(x,y) 像素点 经纬度(lat,lng) 像素点转换经纬度
isPointInPolygon(latlng,[latlng]) 点(lat,lng),多边形点数组 结果(bool) 判断点是否在多边形
Icon类 getPosition() None String 获取Icon图标位置
getIcon() None String 获取Icon图标图片地址
clear() None None 清理加载的图标
setIcon(path,?size,?anchor) 图片相对路径;width,height;x,y None 设置图标图片
drawSign(optionObj) 配置对象:{content:'内容',callBack} 回调 鼠标绘制图标
loadIcon
(id,latlng,imgPath,content,clickBackFun,?size,?anchor)		 标示id,经纬度字符串、图片路径、气泡内容,点击回调id、width,height;x,y None 加载图标
setInfoWindowContent
(content)		 内容 None 设置气泡内容
openInfoWindow
(flag)		 布尔 None 打开或关闭气泡
loadIconAndOpenInfoWindow
(id,latlng,imgPath,content,clickBackFun)		 标示id,经纬度字符串、图片路径、气泡内容,点击回调id None 加载图标并打开气泡
loadCustomIcon(latlng,customHtml) 经纬度字符串,自定义html对象 None 加载用户自定义html覆盖物
CarIcon类 load(gpsObj,clickBackFun) GPS对象{Carlicense:'xxx',GpsLat:120.11,GpsLng:110.11,Direction:240,Status:1},点击回调车牌 String 加载车辆图标
setInfoWindowContent(content) 内容 None 设定气泡内容
clear() None None 清理车辆图标
setZIndex(number) 层数字 None 设置车连图标显示层级
setBorderColor(hexColor) 16进制颜色(默认‘#9C9C9C’) None 设置车辆边框颜色
setPosition(gps) GPS对象 None 设置车辆图标位置
getCenter() None latlng|String 获取车辆图标经纬度位置字符串
Line类 drawLine() 绘制路线
loadLine(lineData) "lat,lng;lat,lng" None 加载路线
addHandlerEvent(name,backfun) name为操作的方式，比如"click",backfun为回调函数 回调内容为操作的点的（lat,lng）经纬度 为路线增加事件
getPath() None "lat,lng;lat,lng" 返回路线的经纬度
setPath(latlng) latlng是经纬度字符串"lat,lng;lat,lng" None 重新加载路线
getDistance(unit) unit测量单位(k=km,m=mi),不填为km None 在绘制路线时测量距离
clear() None None 清除地图上的路线
moveAndZoom() None None 移动到视窗并缩放
Fence类 drawFence() 绘制围栏
getPath() None "lat,lng;lat,lng" 返回围栏的点的坐标集合
setPath(latlng) "lat,lng;lat,lng" None 重新绘制围栏参数为围栏的坐标集合
loadFence(latlng) "lat,lng;lat,lng" None 加载围栏
loadFenceAndEdit(latlng) "lat,lng;lat,lng" None 加载可编辑的围栏
clear() None None 清除地图上的围栏
moveAndZoom() None None 移动到视窗并缩放
Circle类 drawCircle() 绘制圆
setPath(latlng,r) latlng为圆心坐标，r为半径大小 None 重新绘制圆
getPath() None "lat,lng;R"返回圆形的圆心位置和半径大小 返回圆的位置
loadCircle(latlng,r) latlng为圆心位置，r为半径大小 None 加载圆
clear() None None 清除地图上的圆
Rectangle类 drawRectangle() 绘制圆矩形
setPath(latlng) latlng为左上角和右下角的坐标（lat1,lng1;lat2,lng2） None 重新绘制矩形
getPath() None "lat1,lng1;lat2,lng2" 获取矩形的左下角和右上角的坐标
clear() None None 清除地图上的矩形
loadRectangle(latlng) latlng为左下角和右上角的坐标的字符串"lat1,lng1;lat2,lng2" None 在地图上加载矩形
